# 📖 درباره این ریپازیتوری

این ریپازیتوری برای آمادگی سریع مصاحبه ورودی کارشناسی ارشد **Computer Science** در دانشگاه **University of Milan (UNIMI)** ساخته شده است.

هدف این پروژه جایگزین کردن دوره‌های دانشگاهی نیست؛
بلکه کمک می‌کند تا:

* مفاهیم پایه علوم کامپیوتر را سریع مرور کنید
* برای پاسخ‌های شفاهی آماده شوید
* سوالات رایج مصاحبه را تمرین کنید
* اعتمادبه‌نفس بیشتری برای مصاحبه داشته باشید
* نقاط ضعف خود را برای مرور سریع پیدا کنید

این راهنما مخصوصاً برای افرادی مفید است که:

* مدت زیادی از درس فاصله گرفته‌اند
* از رشته یا زمینه متفاوتی وارد کامپیوتر می‌شوند
* زمان کمی برای آمادگی دارند
* دنبال توضیحات کوتاه و کاربردی هستند

---

# 🎯 مصاحبه معمولاً چه چیزهایی را بررسی می‌کند؟

بر اساس تجربه دانشجوها و موضوعات رایج، مصاحبه معمولاً یک بررسی کوتاه شفاهی از مفاهیم پایه علوم کامپیوتر است.

سوالات معمولاً شامل این موارد هستند:

* یک سوال ساده برنامه‌نویسی یا الگوریتم
* یک مفهوم دیتابیس یا SQL
* یک سوال از سیستم‌عامل یا شبکه
* یک مفهوم از معماری کامپیوتر
* یک سوال ساده ریاضی یا آمار

سوالات معمولاً:

* پایه‌ای هستند
* گسترده هستند
* بیشتر مفهومی‌اند تا حفظی
* روی توانایی توضیح دادن تمرکز دارند

---

# 🧠 بهترین ساختار برای پاسخ دادن

برای بیشتر سوال‌ها می‌توانید از این ساختار ۴ مرحله‌ای استفاده کنید:

## 1️⃣ تعریف

اول مفهوم را خیلی کوتاه تعریف کنید.

## 2️⃣ ایده اصلی

توضیح دهید چرا این مفهوم مهم است یا چگونه کار می‌کند.

## 3️⃣ مثال کوچک

یک مثال ساده واقعی یا فنی بزنید.

## 4️⃣ پیچیدگی / کاربرد / مزایا و معایب

اگر مرتبط بود به موارد زیر اشاره کنید:

* پیچیدگی زمانی
* tradeoff
* کاربرد
* مزایا و معایب

# 🛟 اگر چیزی را فراموش کردید چه کار کنید؟

کاملاً طبیعی است که در استرس مصاحبه بخشی از مطالب را فراموش کنید.

به‌جای سکوت، سعی کنید روند فکر کردن خود را توضیح دهید.

جمله‌های مفید:

```text id="persian-phrase-1"
"تعریف دقیقش الان یادم نیست، ولی از نظر مفهومی یعنی ..."
```

```text id="persian-phrase-2"
"اجازه بدید مرحله‌به‌مرحله در موردش فکر کنم."
```

```text id="persian-phrase-3"
"ایده اصلی اینه که ... و یک مثال معمولش ..."
```

```text id="persian-phrase-4"
"ممکنه جزئیات پیاده‌سازی رو کامل یادم نباشه، ولی tradeoff اصلی اینه که ..."
```

معمولاً مصاحبه‌کننده‌ها بیشتر به این موارد اهمیت می‌دهند:

* قدرت تحلیل
* درک مفاهیم
* توانایی توضیح دادن

نه حفظ بودن متن کتاب.
